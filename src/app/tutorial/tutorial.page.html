Tutorial

Users

Users can only be created by admins. Currently we have pre-defined users of which you can choose to experience ReadingPal. Choose from the Teachers or Students lists below to login as that user.

User Card

The user card has the basic information about the current user. A welcome message, their profile picture, and for students a few of their awards.

Navigation

Users can navigate to various pages with by selecting one of the options. (options vary for student and teacher)

Teachers

Teachers Page Descriptions:

  • Classes: As a teacher you will be directed here upon login. You are able to view your classes, groups, students, and reporting from this page.
  • Lessons: You are able to browse all the lessons in the catalog as well as view and edit your lessons. You also have the ability to create new lessons.
  • Books: In books you have the ability to browse and open any book in the ReadingPal catalog. We give access to teacher to be able to upload epubs to this catalog.
  • Awards: Awards are central to incentivize leaning on ReadingPal. You have access to browse all public awards in the catalog as well as add your own awards.
  • Messages: Messages is for communicating between teacher an student. Each message is in the context of a annotation. An annotation is a span of text from a reading in a book. Any lesson a teacher creates a student taking the lesson can ask questions within that lesson. These messages populate here.

Students

Students Page Descriptions:

  • Home: As a Student you will be directed here upon login. Home page is where all assignments will be displayed. This includes: current, past, and overdue assignments.
  • Lessons: You are able to browse all the lessons in the catalog. You are able to open the lesson and complete the tasks.
  • Library: In books you have the ability to browse and open any book in the ReadingPal catalog.
  • Reports: In the reports page you will be able to look at various results from your use of ReadingPal.
  • Messages: The messages page is designed for the student to be able to see and respond to messages created as questions to the teacher. Students are also able to see messages within the reader in the corresponding lesson.
  • Badges: Badges contains a collections of the badges you have earned and allows you to browse all available badges.

Reader

Reader Basic Features

  • Navigation: Navigation consists of paging left and right by using the arrows on either side of the text. (See Table of contents for other options)

  • Font Size: User can change the size of the text to fit their comfort level.

  • Font Family: User can change the font to from a selection of fonts.

  • Font Style: User can change the font style from normal to italic.

  • Font Weight: User can change the font weight from normal to bold.

Reader Annotations

  • Annotate Text: User can highlight passages. Add notes to these passages.

  • Ask Questions: Students have the ability to ask questions to their teacher about sections of text if they are in a reading from a lesson. They can do this within any Annotation.

  • Public Notes: Teachers can add notes to readings that students are able to see while taking a lesson. All notes not marked public are private by default.

Reader Settings

  • Change reader settings This is where the reader will set all desired reading settings for the the current book. Each book will have their own settings.

  • Dark Mode: User can choose between light and dark mode.

Reader Table of Contents

  • Table of Contents: User can navigate by chapter.

Reader Notes

  • Notes: User can view their notes and if in a lesson their teachers notes. They are able to navigate to the notes. As well as hide information about the notes.

Reader Quick Menu

  • Font size: User can change font size

  • Highlight Color: User can select highlight color and update their highlight color palette.

  • Auto Highlight: All selections will be highlighted. Allows for quick highlighting

  • Auto Delete Annotations: All annotations tapped will be deleted. Allows for quick deletion.
